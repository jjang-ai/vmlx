Without prefix caching:
- User 1's request: Compute KV cache for entire document + question

With prefix caching:
- User 1's request: Compute KV cache for document, store in prefix cache
- User 2's request: Retrieve document's KV cache from prefix cache, only compute for new question

Result: User 2 saves computation proportional to document length
