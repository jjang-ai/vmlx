    ```python
    def find_dupes(lst):
        dupes = []
        for i in range(len(lst)):
            for j in range(i+1, len(lst)):
                if lst[i] == lst[j] and lst[i] not in dupes:
                    dupes.append(lst[i])
        return dupes
    ```
    *   **Goal:** Find duplicate elements in a list.
    *   **Logic:** Nested loops comparing every pair `(i, j)` where `j > i`. If `lst[i] == lst[j]`, it checks if `lst[i]` is already in `dupes`. If not, it appends it.
    *   **Input:** `lst` (assumed to be a list).
    *   **Output:** A list of duplicate values found.
