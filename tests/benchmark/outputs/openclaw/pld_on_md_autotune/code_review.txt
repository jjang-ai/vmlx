    ```python
    def find_dupes(lst):
        dupes = []
        for i in range(len(lst)):
            for j in range(i+1, len(lst)):
                if lst[i] == lst[j] and lst[i] not in dupes:
                    dupes.append(lst[i])
        return dupes
    ```

    *   **Logic:**

        *   Checks if `lst[i] == lst[j]`.
        *   Checks if the found duplicate is already in `dupes` list.
        *   Appends to `dupes` if not present.
    *   **Output:** A list of unique duplicate values found.
