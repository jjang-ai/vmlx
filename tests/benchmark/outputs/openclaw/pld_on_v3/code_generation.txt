Problem: Write a Python function that reverses a linked list. Include the Node class definition and a short example in __main__.

Approach: - `prev`: Points to the previous node (initially `None`, becomes the new head)
- `curr`: Points to the current node being processed (initially `head`)
- `temp`: Temporarily stores the next node before we break the link

Key considerations:
- Args:
        head: The head node of the linked list
        
    Returns:
        The new head node of the reversed linked list
        
    Time Complexity: O(n) - single pass through the list
    Space Complexity: O(1) - only using a constant number of pointers
    """
    Edge case: Empty list or single node
    if not head or not head.next:
        return head
    
    prev = None
    curr = head
    
    while curr:
        temp = curr.next  Store next node before breaking the link
        curr.next = prev  Reverse the pointer
        prev = curr       Move prev forward
        curr = temp       Move curr forward
    
    return prev  prev is now the new head
