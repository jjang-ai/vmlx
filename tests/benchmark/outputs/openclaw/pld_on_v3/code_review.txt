**Function Analysis:**
```python
def find_dupes(lst):
    dupes = []
    for i in range(len(lst)):
        for j in range(i+1, len(lst)):
            if lst[i] == lst[j] and lst[i] not in dupes:
                dupes.append(lst[i])
    return dupes
```

- The function finds duplicate elements in a list
- Outer loop iterates through each element
- Inner loop compares each element with subsequent elements
- When a match is found and not already in dupes, it's added
- Edge cases:

  - No duplicates → returns [] ✓
  - All same elements → returns [element] ✓
  - Multiple different duplicates → returns all of them ✓
