    ```python
    def find_dupes(lst):
        dupes = []
        for i in range(len(lst)):
            for j in range(i+1, len(lst)):
                if lst[i] == lst[j] and lst[i] not in dupes:
                    dupes.append(lst[i])
        return dupes
    ```

    *   **Edge Cases:** Empty list, list with no duplicates, list with all duplicates, unhashable types (though `in` operator works for lists, it's slow), order preservation (it preserves the order of first occurrence of the duplicate).
