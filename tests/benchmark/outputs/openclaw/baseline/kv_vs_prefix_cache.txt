# KV Cache vs. Prefix Cache: A Detailed Comparison

## Overview

Both KV cache and prefix cache are memory optimization techniques used in transformer-based language models, but they serve different purposes and operate at different levels of the inference pipeline.

---

## KV Cache

### What It Is:
The **KV (Key-Value) cache** is a fundamental mechanism that stores the computed key and value vectors from previous tokens **during the generation of a single sequence**.
