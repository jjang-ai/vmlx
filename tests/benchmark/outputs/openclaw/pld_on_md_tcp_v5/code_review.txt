    ```python
    def find_dupes(lst):
        dupes = []
        for i in range(len(lst)):
            for j in range(i+1, len(lst)):
                if lst[i] == lst[j] and lst[i] not in dupes:
                    dupes.append(lst[i])
        return dupes
    ```
    *   **Goal:** Find duplicate elements in a list.
    *   **Logic:** Nested loops comparing every pair. If a match is found and the element isn't already in `dupes`, add it.
    *   **Complexity:** $O(N^2)$ for the loops, plus $O(N)$ for the `in` check on `dupes` (worst case), making it potentially $O(N^3)$ if `dupes` grows large, though typically $O(N^2)$ dominates.
