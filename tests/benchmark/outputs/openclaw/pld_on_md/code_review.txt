        ```python
        def find_dupes(lst):
            dupes = []
            for i in range(len(lst)):
                for j in range(i+1, len(lst)):
                    if lst[i] == lst[j] and lst[i] not in dupes:
                        dupes.append(lst[i])
            return dupes
        ```

        *   Checks if `lst[i]` is already in `dupes` list.
        *   If both true, appends to `dupes`.
    *   **Correctness:**

        *   Does it avoid adding the same duplicate value multiple times? Yes, `lst[i] not in dupes` check handles this.
