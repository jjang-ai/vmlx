**Function Analysis:**
The function `find_dupes(lst)` finds duplicate elements in a list.

**Algorithm:**
- Uses nested loops to compare each pair of elements
- Outer loop: i from 0 to len(lst)-1
- Inner loop: j from i+1 to len(lst)-1
- If lst[i] == lst[j] and lst[i] not already in dupes, add it

**CORRECTNESS:**
Let me trace through examples:
- `find_dupes([1, 2, 3])` → `[]` ✓
- `find_dupes([1, 2, 1, 2])` → `[1, 2]` ✓
- `find_dupes([1, 1, 1, 2, 2])` → `[1, 2]` ✓
- `find_dupes([])` → `[]` ✓
- `find_dupes([1])` → `[]` ✓

The function correctly:
1. Finds all duplicate values
2. Returns each duplicate only once (due to `lst[i] not in dupes` check)
3. Handles edge cases (empty list, no duplicates, all same elements)

**PERFORMANCE:**
- Time complexity: O(n²) for nested loops + O(n) for each `not in dupes` check (which is O(d) where d is number of dupes)
- Overall: O(n²) worst case
- Space complexity: O(d) where d is number of duplicates

This can be improved to O(n) using a set or Counter.
