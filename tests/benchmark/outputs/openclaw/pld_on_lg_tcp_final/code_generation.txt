Problem: Write a Python function that reverses a linked list. Include the Node class definition and a short example in __main__.

Approach: - `prev`: Points to the previous node (initially `None`)
- `current`: Points to the current node being processed (initially `head`)
- `next_temp`: Temporarily stores the next node before we break the link

Key considerations:
- Args:
        head: The head node of the linked list to reverse.
    
    Returns:
        The new head node of the reversed linked list.
    
    Time Complexity: O(n) - we visit each node exactly once
    Space Complexity: O(1) - only using a constant number of pointers
    """
    Edge case: Empty list or single node - already "reversed"
    if not head or not head.next:
        return head
    
    Initialize pointers
    prev = None
    current = head
    
    Traverse the list and reverse each node's pointer
    while current:
        Store the next node before we break the link
        next_temp = current.next
        
        Point current node to previous node (reverse the link)
        current.next = prev
        
        Move prev and current one step forward
        prev = current
        current = next_temp
    
    prev is now pointing to the new head (last node of original list)
    return prev

## Architectural Decisions & Reasoning

### 1. **Node Class Design**
- Using a simple class-based approach with `value` and `next` attributes
- Python's `__init__` allows flexible initialization (value required, next optional)
- Added `__repr__` for easy debugging and visualization

### 2. **Algorithm Choice: Iterative vs Recursive**
I chose the **iterative approach** because:
- **Space efficiency**: O(1) extra space vs O(n) stack space for recursion
- **No stack overflow risk**: Important for very long lists
- **Better performance**: No function call overhead
- **Production-ready**: More robust for real-world use

### 3. **Three-Pointer Technique**
The iterative reversal uses three pointers:
- `prev`: Points to the previous node (initially `None`)
- `current`: Points to the current node
