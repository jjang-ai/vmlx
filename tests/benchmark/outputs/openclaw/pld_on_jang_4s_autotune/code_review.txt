    ```python
    def find_dupes(lst):
        dupes = []
        for i in range(len(lst)):
            for j in range(i+1, len(lst)):
                if lst[i] == lst[j] and lst[i] not in dupes:
                    dupes.append(lst[i])
        return dupes
    ```
    *   **Logic:** It iterates through every pair of indices `(i, j)` where `j > i`. If `lst[i] == lst[j]`, it checks if `lst[i]` is already in the `dupes` list. If not, it adds it.
    *   **Functionality:** It finds duplicate values in a list.
    *   **Edge Cases:** Empty list, list with no duplicates, list with all same elements, list with multiple different duplicates.
