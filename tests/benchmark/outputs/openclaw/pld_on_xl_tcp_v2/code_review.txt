    ```python
    def find_dupes(lst):
        dupes = []
        for i in range(len(lst)):
            for j in range(i+1, len(lst)):
                if lst[i] == lst[j] and lst[i] not in dupes:
                    dupes.append(lst[i])
        return dupes
    ```
    *   **Logic:** It iterates through every pair of indices `(i, j)` where `j > i`. If elements match and the element isn't already in `dupes`, it adds it.
    *   **Correctness:** Does it find duplicates? Yes. Does it handle edge cases? Empty list -> returns `[]`. List with no dupes -> returns `[]`. List with multiple dupes (e.g., `[1, 1, 1]`) -> `i=0, j=1` adds `1`. `i=0, j=2` checks `1` in `dupes` (True), skips. `i=1, j=2` checks `1` in `dupes` (True), skips. Result `[1]`. This seems correct for finding *unique* duplicate values.
